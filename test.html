从运动的研究到微积分的创立

用微积分解决四类问题 1. 求瞬时速度。 --> 导数 2. 求曲线的切线。 --> 微分 3. 求函数的最大值和最小值。 --> 定积分 4. 求曲线围成的面积（体积）、求曲线的长度。 --> 二重积分

导数和微分 1. 导数：$f' (x) = \lim{\Delta x \to 0} \frac{f (x+\Delta x) - f (x)}{\Delta x}$ 2. 微分：$\Delta y = f(x0 + \Delta x) - f(x_0), \Delta y = A \cdot \Delta x + o(\Delta x),\Delta y = dy, \Delta x = dx \Rightarrow dy = A \cdot dx$

极限的定义 1. 数列极限：$对任意的 \epsilon > 0, 存在 m \in \mathbb{N}, 使得当 n > m 时，有 |an - a| < \epsilon$，记作 $\lim{n \to \infty} an = a$ 1. 条件：数列 an 是实数数列，N 是正整数。在数列中 n 显然是正整数 2. 满足 $|an - am| < \epsilon$ 的数列称为柯西数列，实数数列有极限的充分必要条件是实数数列是柯西数列 3. 只要实数数列是柯西数列就有极限，不是柯西数列就没有极限（==实数系的完备性==） 2. 函数极限：$对任意的 \epsilon > 0, 存在 \delta > 0 使得当 0 < |x - x0| < \delta 时，有 |f(x) - A| < \epsilon$，记作 $\lim{\Delta x \to 0} f(x) = A$ 1. 说明：$函数 f(x) 在点 x0附近有定义，0 < |x - x0| < \delta 表示一种 x 无限趋近 x_0的过程，A是极限值$

模运算

又称同余运算，一个数 a 除以 n 余数为 m，就可以说是 $a \bmod n = m$。 如果 a 和 b 模 n 得到的余数相同，称 a 和 b 同余，记作 $a \equiv b (\bmod n)$ 如果有两个正整数 a 和 b，并且二数的差值 a-b 为 n 的整数倍数，就可以说 a 和 b 在模 n 下同余。 如果 a 和 b 同余，那么可以表示为 $a = kn + r, b = ln + r$，其中 k 和 l 是整数，r 是模 n 下的共同余数，那么 $a - b = kn + r - ln - r = (k - l)n$，可得到 a-b 是 n 的倍数。 模 m 的剩余类环：在整数集合中，以模 m 进行等价划分后得到的集合。每个等价类由与模 m 同余的整数组成。

充分条件：一个命题的"充分条件"表示该命题成立所必需的条件。如果一个命题 P 是一个命题 Q 的充分条件，那么 Q 成立是 P 成立的前提。换句话说，如果 P 是 Q 的充分条件，那么在 Q 成立时，可以推导出 P 成立。形式上表示为："如果 Q，则 P"，或者 "Q -> P"

必要条件：一个命题的"必要条件"表示该命题成立所必需的条件。如果一个命题 P 是一个命题 Q 的必要条件，那么 P 成立是 Q 成立的前提。换句话说，如果 P 是 Q 的必要条件，那么在 P 成立时，可以推导出 Q 成立。形式上表示为："如果 P，则 Q"，或者 "P -> Q"

素数：也称质数，在大于1的自然数中，除了1和自身之外没有其他正因数（约数）（除了1和该数本身以外没有其他整数可以整除它）的数。换句话说，素数是只能被1和它自己整除的正整数。 合数：合数是大于1的自然数，除了1和它本身之外，还有其他正整数可以整除它。换句话说，合数有至少三个正因数。

奇数：奇数是指除以2余数为1的整数。可以表示为 "2n+1" 的形式。 偶数： 偶数是指可以被2整除的整数，即其余数为0。可以表示为 "2n" 的形式。

因数：如果整数a除以整数b(b≠0) 除得的商正好是整数而没有余数，我们就说a能被b整除，或b能整除a。a称为b的倍数，b称为a的约数。例如 4 的因数有 1，2，4。

`

自然数：是计量物体个数的整数，包括正整数和 0 ，0 是最小的自然数。 整数：小数位为 0，能被 1 整除的数，包括正整数，0，负整数。 有理数：包括整数、小数（有限小数和无限循环小数）、分数（两个整数之比）。 无理数：无限不循环小数，例如 PIE。

实数：相对于虚数而言，是有理数和无理数的总称。 虚数：虚数是一个数学上的概念，它定义为 i，其中 i 是虚数单位，满足 i^2=−1 。虚数通常表示为 bi，其中 b 是实数。 复数：复数是实数和虚数的组合，通常表示为 a+bi，其中 a 和 b 是实数，i 是虚数单位。例如 4-7i，4 是实部， -7i 是虚部

实数系的连续性：用一个无限延伸的数轴表示实数系，数轴上的点如果不是有理数，那么就是无理数，有理数+无理数=实数。数轴是连续的那么实数系也是连续的。
